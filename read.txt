A false positive means:

The Bloom filter says "Yes, the item exists"
But in reality, the item is not actually in the set

Example = ["apple", "banana", "cherry"]
input = "grape" Some time bloom filter give true item exit but Acctual item not exit


A false negative means:

The Bloom filter says "No, the item does not exist"
But in reality, the item is actually in the set



| Case           | In Set (DB)  | Bloom Filter Says   | Result           |
| -------------- | ------       | -----------------   | ---------------- |
| True Positive  | ✅           | ✅                 | ✅ Correct        |
| False Positive | ❌           | ✅                 | ❌ Incorrect      |
| False Negative | ✅           | ❌                 | ❌ **Impossible** |
| True Negative  | ❌           | ❌                 | ✅ Correct        |
